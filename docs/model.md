# Математическая модель

## Цель
Оптимизация выбора способа доставки для минимизации итоговой стоимости.

## Переменные
- \( w \) — вес запчасти (кг)
- \( d \) — расстояние доставки (км)
- \( m \) — способ доставки (1, 2, 3)

## Функция стоимости доставки
\[
C_{delivery}(w, d, m) = 
\begin{cases} 
10w + 5d & \text{если } m = 1 \text{ (обычная)} \\
1.5 \times (10w + 5d) & \text{если } m = 2 \text{ (экспресс)} \\
0 & \text{если } m = 3 \text{ (самовывоз)}
\end{cases}
\]

## Целевая функция
Минимизация итоговой стоимости:
\[
F(w, d) = \min_{m \in \{1,2,3\}} \left( C_{base} + C_{delivery}(w, d, m) + C_{insurance} + C_{urgent} - S \right)
\]

## Пример оптимизации
При \( w = 5 \) кг, \( d = 50 \) км:

| Способ | Формула | Стоимость |
|--------|---------|-----------|
| Обычная | (5×10)+(50×5) = 50+250 | 300 руб |
| Экспресс | 300 × 1.5 | 450 руб |
| Самовывоз | 0 | 0 руб |

**Оптимальный способ:** Самовывоз (если возможен)

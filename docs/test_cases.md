# Тестовые сценарии

## Тест-кейс 1: Стандартный расчёт (обычная доставка)
| Параметр | Значение |
|----------|----------|
| Название | Тормозные колодки |
| Цена | 2500 руб |
| Вес | 1.5 кг |
| Расстояние | 30 км |
| Доставка | 1 (обычная) |
| Срочность | Нет |
| Страховка | Нет |
| Промокод | Нет |

**Ожидаемый результат:**
- Доставка = (1.5 × 10) + (30 × 5) = 15 + 150 = 165 руб
- Итого = 2500 + 165 = 2665 руб

## Тест-кейс 2: С промокодом SALE10
| Параметр | Значение |
|----------|----------|
| Цена | 2500 руб |
| Вес | 1.5 кг |
| Расстояние | 30 км |
| Доставка | 1 (обычная) |
| Промокод | SALE10 |

**Ожидаемый результат:**
- Доставка = 165 руб
- Сумма до скидки = 2665 руб

- Итого = 2665 - 266.5 = 2398.5 руб

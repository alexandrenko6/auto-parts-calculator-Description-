## 1. Техническое задание

### 1.1 Назначение программы
Программа для расчёта итоговой стоимости автозапчасти с учётом доставки, скидок и дополнительных услуг.

### 1.2 Входные данные
- Название запчасти
- Базовая цена (руб)
- Вес (кг)
- Расстояние (км)
- Способ доставки (1-3)
- Промокод (опционально)
- Страховка (да/нет)
- Срочная доставка (да/нет)

### 1.3 Выходные данные
- Детализированный расчёт с разбивкой по статьям
- Итоговая стоимость
- Сохранение в лог-файл

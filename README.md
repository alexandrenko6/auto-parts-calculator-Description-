# auto-parts-calculator-Description-
# Калькулятор стоимости автозапчасти с доставкой

## Назначение программы
Программа предназначена для расчёта итоговой стоимости автозапчасти с учётом:
- базовой цены запчасти
- стоимости доставки в зависимости от веса, расстояния и способа доставки
- возможных скидок по промокодам
- дополнительных услуг (страховка, срочная доставка)

## Входные данные
| Параметр | Тип | Описание |
|----------|-----|----------|
| Название запчасти | строка | Наименование запчасти |
| Базовая цена | число (руб) | Цена запчасти без доставки |
| Вес запчасти | число (кг) | Вес для расчёта доставки |
| Расстояние доставки | число (км) | Расстояние от склада до клиента |
| Способ доставки | 1-3 | 1-обычная, 2-экспресс, 3-самовывоз |
| Промокод | строка | Опционально, для скидки |
| Страховка | да/нет | +5% от суммы |
| Срочная доставка | да/нет | +500 руб |

## Выходные данные
- Базовая цена
- Стоимость доставки
- Скидка (если применена)
- Стоимость страховки (если выбрана)
- Плата за срочность (если выбрана)
- **Итоговая стоимость**

## Формула расчёта
# Доступные промокоды
| Промокод | Скидка |
|----------|--------|
| SALE10 | 10% |
| SALE20 | 20% |
| WELCOME | 5% |
| BLACKFRIDAY | 25% |
